# computer science vocabulary

## Introduction

I created vocabulary pages for mathematics and chess

I thought I would do the same for computer science

In the table below, I'll define some important terms from computer science

## Computer science vocabulary

| Word | Definition |
| ---- | ---------- |
| Algorithm | A list of instructions |
| ASCII | (1) An acronym meaning American Standard Code for Information Interchange (2) A 7-bit character encoding system |
| Binary | A positional number system in base two |
| Ciphertext | (1) An encrypted message (2) Encrypted information |
| Plaintext | Unencrypted information |
| Processor | A chip that executes instructions |
